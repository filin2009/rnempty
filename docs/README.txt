# Documentov

## Введение

Documentov - это свободная система автоматизации документно-ориентированных бизнес-процессов, разработанная на основе ecommerce CMS OpenCart (www.opencart.com).

## Сообщение об ошибках

Если вы обнаружили ошибку:

1. Попробуйте найти решение на нашем форуме https://www.documentov.com/forum

2. Если решение не найдено создайте новую тему.
